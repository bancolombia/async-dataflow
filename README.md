## How to use

### Requirements

- Elixir >= 1.10
- Mix

### Install
```elixir
mix deps.get
mix compile
```
### Configuration

Open and edit config/dev.exs and config/config.exs Files to configure

### Run
In the shell:
```elixir
iex -S mix 
or
iex  --sname node1@localhost -S mix
```
